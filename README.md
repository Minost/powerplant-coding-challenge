# powerplant-coding-challenge

## Usage

In the project directory, launch the Flask application to handle the incoming production plan requests:

```bash
python main.py
```

Then you will be able to call the endpoint to plan the production based on the provided json payload:

```bash
localhost:8888/productionplan
```

For example, you can curl the endpoint with a provided example payload:
